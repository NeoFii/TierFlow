# 什么是 MindRouter？

MindRouter 是一个 **AI 模型智能路由平台**。我们接入了 OpenAI、Anthropic、Google、DeepSeek、Mistral 等主流大模型供应商的 API，通过统一的接口为开发者提供服务。

核心价值：**在不降低输出质量的前提下，通过智能路由将 API 调用成本降低最高 60%。**

## 核心问题

当前使用大模型 API 的开发者面临几个普遍痛点：

| 痛点 | 说明 |
|------|------|
| **成本高昂** | 旗舰模型（GPT-4o、Claude Opus）单价高，但大量请求其实不需要这个级别 |
| **选择困难** | 模型越来越多，每个模型擅长的领域不同，难以针对每个任务选择最优模型 |
| **供应商锁定** | 每个厂商 API 格式不同，切换成本高，一旦绑定就很难迁移 |
| **可用性风险** | 单一供应商出现故障或限流时，业务直接中断 |

## MindRouter 如何解决

```
用户请求 → MindRouter 路由引擎 → 分析任务复杂度
                                      ↓
                              ┌───────────────────┐
                              │  路由决策引擎      │
                              │                   │
                              │  · 任务复杂度评估  │
                              │  · 模型能力匹配    │
                              │  · 成本优化计算    │
                              │  · 延迟感知调度    │
                              └───────┬───────────┘
                                      ↓
                    ┌─────────┬───────┴────────┬──────────┐
                    ▼         ▼                ▼          ▼
                 OpenAI   Anthropic        Google     DeepSeek
                GPT-4o    Claude 3.5     Gemini 2.0   DeepSeek-V3
                GPT-4o    Sonnet          Flash        Chat
                mini      Haiku          Pro
```

1. **统一 API**：兼容 OpenAI 格式，已有代码改一行 `base_url` 即可接入
2. **智能路由**：根据请求内容自动评估复杂度，匹配性价比最优的模型
3. **故障转移**：某个供应商出问题时，自动切到同级别备用模型
4. **成本透明**：实时仪表盘展示每个请求的模型选择和费用明细

## 路由决策依据

MindRouter 基于以下多维指标进行路由决策：

### 性能指标

| 指标 | 说明 | 更新频率 |
|------|------|----------|
| **P90 延迟** | 各模型 API 的第 90 百分位响应时间 | 实时 |
| **可用率** | 近 24 小时内的 API 成功率 | 每分钟 |
| **吞吐量** | 当前时段的 tokens/s 处理速度 | 实时 |

### 成本指标

| 模型 | 输入价格 ($/1M tokens) | 输出价格 ($/1M tokens) | 等级 |
|------|----------------------|----------------------|------|
| GPT-4o | $2.50 | $10.00 | 旗舰 |
| Claude 3.5 Sonnet | $3.00 | $15.00 | 旗舰 |
| Gemini 2.0 Flash | $0.10 | $0.40 | 高性价比 |
| GPT-4o mini | $0.15 | $0.60 | 高性价比 |
| DeepSeek-V3 | $0.27 | $1.10 | 高性价比 |

> 价格为虚拟示例数据，实际价格以各厂商官方公告为准。

## 适用场景

- **SaaS 产品**：用户量大、请求种类多，大量简单查询不需要旗舰模型
- **AI Agent / 工作流**：多步骤任务链中，不同步骤复杂度不同，可分别路由
- **内容生成平台**：翻译、摘要等标准任务用轻量模型，创意写作用旗舰模型
- **企业内部工具**：控制 API 预算的同时保证关键任务的输出质量
